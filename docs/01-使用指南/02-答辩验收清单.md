## 四、与原型差异说明（答辩口述用）


| 项               | 说明                                                             |
| --------------- | -------------------------------------------------------------- |
| Element Plus 控件 | 设计文档允许 Vue3 + Element Plus，下拉/时间选择与原型原生 select 外观略有不同，**交互一致** |
| 管理端反馈页          | 设计文档 A10 要求，原型 sidebar 无此项，**以文档为准保留**                         |
| 暂离按钮            | V1.1 已实现，原型 mock 未展示按钮，**功能优于原型**                              |
| 修改密码            | 后端接口待 V1.2，设置页保留入口并提示「待接入」                                     |

## 五、常见问题


| 现象       | 处理                                      |
| -------- | --------------------------------------- |
| UI 仍是旧版  | 执行 `.\scripts\build-frontend.ps1` 后重启后端 |
| 8080 被占用 | 关闭旧 Java 进程或换端口                         |
| 扫码失败     | 管理员端手动粘贴学生 QR token                     |
