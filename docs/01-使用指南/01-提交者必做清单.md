# 01 — 提交者必做清单（仅保留你必须做的事）

> **给谁看**：提交仓库、同步数据给组长的同学（你）  
> **工作目录**：`CSRRMupdate`（含 `pom.xml` 的目录）  
> **组长怎么做**：不要看本文，请看项目根目录 **[组长复现指南.md](../../组长复现指南.md)**。

---

## 你必须做的只有两件事

### 一、本机继续开发 / 测试时

1. **保证 MySQL 已启动**（服务名一般为 MySQL80）。  
2. **配置密码**（仅本机，不提交 Git）：  
   - 复制 `src/main/resources/application-local.properties.example`  
   - 为 `application-local.properties`  
   - 填写 `spring.datasource.password=你的密码`  
3. **启动**：

```powershell
cd D:\SchoolWorkPlace\Database\CSRRMupdate
.\mvnw.cmd spring-boot:run
```

4. 浏览器打开 **http://localhost:8080** 自测。

改前端界面后需重新打包再启动：

```powershell
.\scripts\build-frontend.ps1
.\mvnw.cmd spring-boot:run
```

---

### 二、把「当前完整数据 + 代码」交给组长（每次要同步前做一次）

测试产生新预约、签到、反馈等之后，若希望组长看到**与你现在一致**的库和界面，按顺序执行：

#### 步骤 1：导出并检查

```powershell
cd D:\SchoolWorkPlace\Database\CSRRMupdate
.\scripts\prepare-full-delivery.ps1
```

（脚本会 `mysqldump` 更新 `docs/06-部署配置/` 下三个 `.sql`，并检查 `uploads`、`static`。）

若提示 mysqldump 密码错误，可显式传密码：

```powershell
.\scripts\export-database-for-git.ps1 -Password 你的MySQL密码
```

#### 步骤 1.5：自测（推荐，约 2 分钟）

```powershell
.\scripts\verify-leader-startup.ps1 -MySqlPassword 你的MySQL密码
```

通过后会显示 `[OK] http://localhost:8080 returned 200`。

#### 步骤 2：提交并推送

```powershell
git add docs/06-部署配置/schema.sql docs/06-部署配置/data.sql docs/06-部署配置/database-full.sql
git add docs/06-部署配置/delivery-manifest.json docs/06-部署配置/init-shared-mysql.sql
git add 组长复现指南.md "docs/01-使用指南/01-提交者必做清单.md"
git add scripts/setup-after-clone.ps1 scripts/prepare-full-delivery.ps1
git add uploads/material uploads/layout uploads/common
git add src/main/resources/static
git add -A
git status
git commit -m "chore: 全量交付快照（数据库+uploads+组长复现指南）"
git push origin v1.3-harrydai4
```

#### 步骤 3：通知组长

把下面两句话发给组长即可：

1. 仓库地址：https://github.com/Harrydai4/CampusStudyRoomReservationManagementSystem  
2. 告诉组长：**Code → Download ZIP** → 解压 → 双击 **`组长一键启动.vbs`**（**不需要** git、不需要切换分支）  
3. 解压后先看 **`00-组长请看这里.txt`**，双击 **`组长一键启动.vbs`**；详见 **`组长复现指南.md`**

## 不要提交到 Git 的文件

| 文件 | 原因 |
|------|------|

| `application-shared.properties` | 含共用库密码 |
| `target/`、`frontend/node_modules/` | 可重新生成 |
