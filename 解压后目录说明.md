# 解压后目录说明（组长 / 验收）

> 解压 ZIP 后，请进入**含 `pom.xml` 的这一层**（项目根目录）。  
> 所有「双击启动」文件都在根目录，不要进到 `src` 或 `docs` 里找。

---

## 一、组长只需关心（项目根目录）


| 文件                  | 用途             |
| ------------------- | -------------- |
| `**00-组长请看这里.txt`** | 3 步快速上手（解压后先看） |
| `**组长一键启动.vbs**`    | **主启动入口**（双击）  |
| `组长一键启动.bat`        | 备用启动（命令行窗口版）   |
| `组长复现指南.md`         | 详细复现、排错、验收清单   |
| 本文件 `解压后目录说明.md`    | 目录结构地图         |


---

## 二、数据库与资源（提交者已打包进 ZIP）


| 路径                                    | 内容                |
| ------------------------------------- | ----------------- |
| `docs/06-部署配置/database-full.sql`      | **整库快照**（启动器自动导入） |
| `docs/06-部署配置/schema.sql`             | 仅表结构（课设审阅用）       |
| `docs/06-部署配置/data.sql`               | 仅数据               |
| `docs/06-部署配置/delivery-manifest.json` | 导出时间、文件清单         |
| `uploads/material/`、`uploads/layout/` | 注册材料、平面图（约 8 个文件） |
| `src/main/resources/static/`          | 已构建前端（无需 npm）     |


---

## 三、脚本目录（一般不用手动点）


| 路径                                  | 用途                       |
| ----------------------------------- | ------------------------ |
| `scripts/setup-after-clone.ps1`     | 命令行一键导入+启动（可选）           |
| `scripts/import-database-local.ps1` | 仅导入数据库（可选）               |
| `scripts/check-env.ps1`             | 检查 JDK / MySQL / 前端（排错用） |


---

## 四、开发相关（组长可忽略）


| 路径              | 内容                           |
| --------------- | ---------------------------- |
| `src/`          | Java 后端源码                    |
| `frontend/`     | Vue 源码（已构建到 static，组长不必 npm） |
| `docs/01-使用指南/` | 提交者、答辩、Git 等文档               |
| `mvnw.cmd`      | Maven 包装（由启动器自动调用）           |


---

## 五、本机生成（不会出现在 ZIP 里）


| 路径                                                | 说明                    |
| ------------------------------------------------- | --------------------- |
| `src/main/resources/application-local.properties` | 本机 MySQL 密码等（启动器自动生成） |
| `target/`                                         | 编译输出（首次启动后产生）         |


---

## 六、常见问题：进错文件夹


| 现象                       | 处理                                                             |
| ------------------------ | -------------------------------------------------------------- |
| 找不到 `组长一键启动.vbs`         | 往上一级找，直到能看到 `pom.xml`                                          |
| 提示缺少 `database-full.sql` | 重新下载完整 ZIP（见 `00-组长请看这里.txt` 直链）                               |
| 解压包名很长                   | 正常，如 `CampusStudyRoomReservationManagementSystem-1.0`，进入该文件夹即可 |
